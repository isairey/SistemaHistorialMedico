# 💊 MediRecord  
## Sistema Inteligente de Recordatorio de Medicamentos

**MediRecord** es una plataforma web diseñada para mejorar la adherencia a tratamientos médicos en adultos mayores mediante la gestión eficiente de medicamentos y el envío de recordatorios automatizados.

---

## 📌 Descripción

El sistema permite a pacientes y cuidadores administrar tratamientos médicos de forma sencilla, asegurando que los medicamentos se tomen en tiempo y forma.

A través de integraciones con servicios de mensajería como **WhatsApp**, MediRecord proporciona alertas oportunas que ayudan a prevenir olvidos y mejorar la salud del usuario.

---

## 🎯 Objetivo

- 💊 Reducir olvidos en la toma de medicamentos  
- 👴 Mejorar la calidad de vida de adultos mayores  
- 🤝 Facilitar el seguimiento por parte de cuidadores  
- 📊 Registrar historial de cumplimiento  

---

## 🚀 Funcionalidades principales

### 💊 Gestión de medicamentos
- Registro de medicamentos  
- Configuración de horarios y dosis  
- Edición y eliminación de tratamientos  

### ⏰ Recordatorios automatizados
- Envío de notificaciones vía WhatsApp  
- Alertas programadas según horarios  

### 👤 Gestión de usuarios
- Perfiles para pacientes  
- Perfiles para cuidadores  

### 🔗 Vinculación paciente-cuidador
- Asociación entre usuarios  
- Supervisión remota del tratamiento  

### 📊 Historial de tomas
- Registro de cumplimiento  
- Seguimiento del tratamiento  

---

## 🧠 Beneficios

- ⏰ Mejora la puntualidad en la medicación  
- 📉 Reduce riesgos por omisión de dosis  
- 📱 Acceso sencillo desde navegador  
- 🤝 Mayor control para familiares y cuidadores  

---

## 🛠️ Tecnologías utilizadas

- **Backend:** PHP 8+  
- **Base de datos:** MySQL / MariaDB  
- **Frontend:** HTML, CSS, JavaScript  
- **Mensajería:** WhatsApp Business API

## ⚙️ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/tuusuario/medirecord.git
```
Acceder al proyecto:
```
cd medirecord
```
Importar la base de datos:
```
Crear una base de datos en MySQL/MariaDB
```
Importar el archivo:
```
medirecord_db.sql
```
Configurar el archivo:
```
config.php
```

- Credenciales de base de datos
- Configuración del servidor
- Configurar WhatsApp Business API
- Integrar credenciales de API
- Definir número de envío
